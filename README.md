Convergent
==========

Simple app to test HTTP currency level

Steps to get started
1. `npm install`
2. Start mock server `node mock.server.js`
3. Initiate the test `node convergent.js`
4. Test result should show as below

```
=================================

=================================

 Runtime: 10000 ms
 interval: 1000 ms

```
